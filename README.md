# テーブル設計

## users テーブル

| Column             | Type   | Options     |
| ------------------ | ------ | ----------- |
| name               | string | null: false |
| email              | string | null: false |
| age                | int    | null: false |
| sex                | int    | null: false |
| prefecture         | string | null: false |

### Association

- has_many :questions
- has_many :comments
